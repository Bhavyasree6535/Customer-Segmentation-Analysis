# Customer Segmentation Analysis

## About the Project
Divided 8,000+ customers into 4 groups based on their buying behavior
to help the business understand which customers need attention.

## Tools Used
- Python (Pandas, NumPy, Matplotlib)
- Jupyter Notebook

## What I Did
- Loaded and cleaned the customer dataset using Pandas
- Grouped customers into 4 categories: High Value, At Risk, Loyal, and New
- Used RFM analysis (how recently, how often, how much customers bought)
- Created charts using Matplotlib to visualize the customer groups
- Found that 22% of customers had not purchased in over 90 days

## What I Found
- The At-Risk group needed immediate attention
- Suggested different marketing strategies for each group
- Retention could improve by 15–20% with targeted approach
